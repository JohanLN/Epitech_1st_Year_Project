Le BSQ est un projet réalisé en première année à Epitech.

Le but de ce projet est de faire un algorithme capable de trouver le plus grand carré dans une map composé de point et avec un ou plusieurs obstacles en forme de "o".

Lorsque l'algorithme trouve le plus grand carré de cette map, il le dessine avec des "x".

Pour compiler le projet utilisez la commande "make". Puis utiliser la commande "./bsq <chemin_vers_la_map>" pour lancer le projet.

Les maps se trouvent dans le dossier "test/moulis_maps". Vous pouvez aussi lancer un test rapide de l'algorithme en exécutant la commande "./test.sh" dans le dossier test.
